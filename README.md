かんたん問診サイト 更新版

更新内容
- 最下部に「医師の話を文字起こし」欄を追加
- Safari と Chrome を自動判定
- Safari の場合は Safari 系として案内文を表示
- Chrome の場合は Chrome 系として案内文を表示
- 対応ブラウザでは Web Speech API 系の音声認識を使用
- 未対応時のために手動貼り付け欄も追加
- 文字起こし結果を「自由メモ」に入れるボタンを追加
- ダークモード時は黄色系文字のまま利用可能

使い方
1. index_safari_chrome_transcript.html をそのままブラウザで開く
2. GitHub Pages に置く場合は index.html にリネームしてアップロード
3. Safari / Chrome で開くとブラウザを自動判定
4. 「マイク開始」で文字起こし開始
5. 必要なら「自由メモに入れる」で問診側に転記

注意
- このサイトはブラウザの音声認識を使います
- iPhone の専用アプリ画面の文字起こし機能そのものを直接呼び出すわけではありません
- ブラウザにより動作差があります
- 未対応の場合は貼り付け欄を使ってください
